# todo-app

<h3> Aplicação que permite ao usuário uma criação de toDo list </h3>
<p>Lista de tarefas que o usuário deve cumprir. Caso o usuário realize essas tarefas, 
ele pode marcar como feito.  </p>

<ul>
Tecnologias
<li>Javascript </li>
<li>Axios </li>
<li>Yarn </li>
<li> React </li>
<li> Redux </li>
</ul>

<h3>Configuração </h3>

<ol>
<li>
Para instalar as dependências :
Yarn 
</li>
<li>
Para iniciar a aplicação :
yarn start 
</li>
</ol>
